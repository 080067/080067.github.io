网站推荐

发表于2023-02-05|更新于2023-02-05|随想
|字数总计:1.2k|阅读时长:4分钟|阅读量:

精选的网站，非常好用，也可以当个备忘录用๑ᵒᯅᵒ๑

1: pixiv

网址：https://pixiv.net
介绍：日本著名的二次元插画网站资源只有你想不到 ps：记得节制一点（小声）
缺点：国内需要翻墙

3: 时琳の个人博客

网址: https://shilinan.top
介绍：此博客分享科技、生活、学习等等值得收藏

4: opeanaiのChatGpt官网

网址：https://chat.opeanai.com
介绍：ChatGPTの官网，可以去玩chatgpt，前提是有账号和密码，有时间会出一期教程

5: facebook官网

网址：https://facebook.com
介绍：国外最大的交流媒体，与推特齐头并进

6:时琳の盘

网址：https://pan.shilinna.top
介绍：博主的个人网盘(可能会寄)，里面会分享一些文章里面需要的东西

7:我告诉你

网址：https://next.itellyou.cn/
介绍：windows镜像下载站(镜像都是纯净的)支持网盘下载与bt种子

8:杏铃のgalgame站

网址：https://g.杏铃.top
介绍：还在为找不到galgame而苦恼吗？不妨试试这个galgame站。

9:archlinux简明手册

网址：https://arch-linux.osrc.com/
介绍：关于archlinux的安装，kde配置，以及美化、软件安装之类的arch Linux使用手册。

10:github

网址：https://github.com
介绍：最大の开源网站，内容十分丰富，进去探索吧(记得开代理)

11:YouTube

网址：https://www.youtube.com/
介绍：国外最大的流媒体网站类似国内B站

12:网易邮箱

网址：https://mail.163.com （https://126.com）
介绍：网易的邮箱登录，查看网站

13:网易云音乐

网址：https://music.163.com
介绍：网易家的听歌网站

15:阿里云

网址：https://aliyun.com
介绍：阿里巴巴集团旗下公司，是全球领先的云计算及人工智能科技公司之一

16:vercel

网址：https://vercel.com/
介绍：vercel 是一个站点托管平台，提供CDN加速，同类的平台有 Netlify 和 Github Pages ，相比之下， vercel 国内的访问速度更快，并且提供 Production 环境和 development 环境，对于项目开发非常的有用的，并且支持持续集成，一次 push 或者一次 PR 会自动化构建发布，发布在 development 环境，都会生成不一样的链接可供预览。

17:网易有道

网址：https://youdao.com
介绍：网易旗下利用大数据技术提供移动互联网应用的子公司，翻译软件可以用它

My free MP3

网址：https://tool.liumingye.cn/music/#/
介绍：myfreemp3 app 是一款简约优质的歌曲 搜索软件 ，简洁的搜索界面，内含多个音乐平台的搜索通道，只需输入关键字即可搜索出你想要的音乐，无损音质高清播放体验，一键加入自己的歌单，喜欢听歌又找不到歌的朋友就来试试吧。

cloudflare

网址：https://www.cloudflare.com/
介绍：Cloudflare 是一家全球最著名的 CDN 加速服务商，提供了免费和付费的网站加速和保护服务。即使是免费版，也提供了比较全面和强大的功能，非常不错。

mcsmanager

网址：https://mcsmanager.com/
介绍：MCSManager是一个针对于mc开服和管理的web面板，基于Node。 MCSM能兼容大量游戏，而不仅限于MC和起源类游戏。 MCSM分为两个主要程序：web面板和deamon守护进程。 web主要负责前端与用户的交互，deamon负责管理真正的服务断进程。

go-cqhttp

网址：https://docs.go-cqhttp.org/
介绍：go-cqhttp是一个基于Go语言开发的，运行在服务器上的机器人框架，它与QQ客户端通过插件，直接进行通信，可以实现对QQ群及私聊的监控、消息发送和自动回复等功能。

alist简明手册

网站：https://alist.nn.ci/zh/
介绍：alist网盘的搭建、挂载教程

frp内网穿透(MC)

网址：https://www.natfrp.com/user/
介绍：frp是一个专注于内网穿透的高性能的反向代理应用，支持 TCP、UDP、HTTP、HTTPS 等多种协议。 可以将内网服务以安全、便捷的方式通过具有公网 IP 节点的中转暴露到公网。

茶杯狐

网址：https://cupfox.app/
介绍：为用户提供热播电视剧，电影、动漫、美剧、韩剧在线观看，想看的视频资源一搜就有，让你通过专业的影视网站搜索来找到喜欢的电影信息，发现好电影，分享电影和在线追剧的乐趣。

次元小镇

网址：https://dimtown.com/
介绍：p站替代品，懂得都懂

gitee

网址：https://gitee.com/
介绍：gitub的国内镜像站(详见github)

github下载加速

网址：https://ghproxy.com/
介绍：可以加快github的下载速度，比较好用
